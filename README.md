# ❤️ Heart Stroke Prediction Model

A machine learning web application that predicts the probability of stroke based on patient health information.

## Features

- Stroke prediction using Machine Learning
- User-friendly web interface
- Data preprocessing
- Feature engineering
- Model evaluation
- Real-time prediction

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- SQLAlchemy
- FastAPI

## Project Structure

```
Heart-Stroke-Predictions-Model/
│
├── data/
├── model/
├── notebooks/
├── app.py
├── requirements.txt
├── README.md
└── LICENSE
```

## Installation

```bash
git clone https://github.com/gurudev-1113/Heart-Stroke-Predictions-Model.git

cd Heart-Stroke-Predictions-Model

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
